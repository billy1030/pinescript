# AKRTS — 使用指南與顏色說明

## 快速開始
1. 開啟 TradingView → Pine Editor → 貼上 `AKRTS.pine` 的內容
2. 點擊 **Add to Chart（加入圖表）**
3. 建議使用 **5分鐘或15分鐘** 線圖（針對 SOXL/SOXS 等槓桿型 ETF 最佳化）

---

## 顏色與視覺元素說明

### 背景色（HTF 趨勢過濾器）
| 顏色 | 含義 |
|---|---|
| **淡藍色背景** | 1小時（HTF）趨勢為**多頭** — 偏向做多 |
| **淡黃色背景** | 1小時（HTF）趨勢為**空頭** — 偏向做空 |
| **無色（透明）** | HTF 趨勢**中性** — 無方向偏好 |

### 核迴歸線（Kernel Lines）
| 線條 | 預設顏色 | 含義 |
|---|---|---|
| **Kernel A（快線）** | 藍色 (#448AFF) | 靈敏趨勢線 — 對價格反應迅速 |
| **Kernel B（慢線）** | 橘色 (#FFAB40) | 穩定趨勢線 — 平滑、移動較慢 |
| **Kernel A 在 Kernel B 上方** | 包絡線填充**綠色** | 整體趨勢為**多頭** |
| **Kernel A 在 Kernel B 下方** | 包絡線填充**紅色** | 整體趨勢為**空頭** |

### 包絡線帶（Envelope Bands）
| 元素 | 顏色 | 含義 |
|---|---|---|
| **上軌** | 淡紅色線 | 壓力 / 超買區域邊界 |
| **下軌** | 淡綠色線 | 支撐 / 超賣區域邊界 |
| **上下軌之間填充** | 綠色或紅色（半透明）| 趨勢方向 — 綠色=多頭，紅色=空頭 |

### 圖表上的信號形狀
| 形狀 | 顏色 | 大小 | 信號 |
|---|---|---|---|
| ▲ 三角形向上 | **綠色** | 正常 | **多頭首次回調** — 趨勢延續進場點 |
| ▼ 三角形向下 | **紅色** | 正常 | **空頭首次回調** — 趨勢延續進場點 |
| ◆ 菱形 | **綠色**（淡） | 極小 | **一般均值回歸做多** — 價格在 -1.5σ |
| ◆ 菱形 | **綠色**（亮） | 正常 | **強力均值回歸做多** — 價格在 -2.5σ（高概率）|
| ◆ 菱形 | **紅色**（淡） | 極小 | **一般均值回歸做空** — 價格在 +1.5σ |
| ◆ 菱形 | **紅色**（亮） | 正常 | **強力均值回歸做空** — 價格在 +2.5σ（高概率）|

### 深入解析：兩種形狀的「賺錢策略」

這兩種形狀分別代表了交易中**「兩種完全不同的策略」**！

#### 1. 綠色/紅色三角形：【順勢上車點】（Pullback 回調）
**這是「最安全、勝率最高」的玩法，適合穩健型玩家。**
* **背後邏輯**：趨勢已經發動了（公車開走了），但它中間一定會踩煞車休息（價格回落）。三角形就是告訴你：「車子暫停了，趕快上車！」
* **▲ 綠色三角形（做多）**：在大漲的過程中，價格稍微跌下來休息一下，然後又準備繼續往上衝。這時候買入。
* **▼ 紅色三角形（做空）**：在大跌的過程中，價格稍微反彈喘口氣，接著準備繼續往下摔。這時候做空。
* **💡 必殺技（配合背景色）**：
  * **淡黃色背景（看多）** ＋ **綠色三角形** ＝ **完美買點！勝率極高！**
  * **淡藍色背景（看空）** ＋ **紅色三角形** ＝ **完美空點！勝率極高！**

#### 2. 綠色/紅色菱形：【極限反彈點】（Mean Reversion 均值回歸）
**這是「抓轉折、刀口舔血」的玩法，適合喜歡刺激和抓最高/最低點的玩家。**
* **背後邏輯**：價格就像一條橡皮筋，拉得太長（漲太多或跌太多），就一定會用力彈回原點！菱形就是告訴你：「橡皮筋快斷了，準備反彈！」
* **◆ 淡色小菱形（小拉扯）**：
  * 橡皮筋被稍微拉長了。不建議這時候進場猜底。但這是個**「警告信號」**，提醒你可以考慮先賣掉一半，入袋為安。
* **◆ 亮色大菱形（極限拉伸，高概率反彈）**：
  * **亮綠色大菱形（超跌）**：市場恐慌跌過頭了，隨時會報復性大漲。可以嘗試**做多**抓反彈。
  * **亮紅色大菱形（超買）**：市場太瘋狂漲過頭了，隨時會發生大崩盤。可以嘗試**做空**抓回落。
* **💡 作戰建議**：抓大菱形反彈叫「富貴險中求」，就算背景顏色不對也可以嘗試。但記得一旦賺錢就要跑得快，而且止損一定要設好！

### 買入/賣出/退出標籤
| 標籤 | 顏色 | 含義 |
|---|---|---|
| **B1 / 50%** | 綠色 ▲ | **第一層買入** — 開倉 50%（匯合度 ≥ 5） |
| **B2 / +25%** | 綠色 ▲（小） | **第二層買入** — 加倉 25%（匯合度 ≥ 7） |
| **B3 / +25%** | 綠色 ▲（小） | **第三層買入** — 最後加倉 25%（匯合度 ≥ 9，總計=100%） |
| **S1 / 50%** | 紅色 ▼ | **第一層賣出** — 開空倉 50%（匯合度 ≥ 5） |
| **S2 / +25%** | 紅色 ▼（小） | **第二層賣出** — 加空倉 25%（匯合度 ≥ 7） |
| **S3 / +25%** | 紅色 ▼（小） | **第三層賣出** — 最後加空 25%（匯合度 ≥ 9） |
| **SL ✕** | 橘色 | **止損** — 平倉 100% |
| **TP ✕** | 綠色 | **止盈** — 平倉 100% |
| **Trail ✕** | 橘色 | **追蹤止損**觸發 — 平倉 100% |
| **Cross ✕** | 橘色 | 核迴歸線交叉反向 — 平倉 100% |
| **Partial XX%** | 淡橘色 | 部分平倉 — 平倉 25% 或 50% |

### 之字線（Zig-Zag）
| 元素 | 顏色 | 含義 |
|---|---|---|
| **藍色點狀對角線** | 藍色 (#2196F3) | 連接擺動高點和低點 — 顯示市場結構 |

### 持倉面板（右上角）
| 欄位 | 顯示內容 |
|---|---|
| **Position（持倉）** | LONG（多頭）/ SHORT（空頭）/ FLAT（空倉），帶顏色標示 |
| **Size（倉位）** | 當前持倉比例 %（0%、50%、75%、100%） |
| **Entry（進場價）** | 當前倉位的進場價格 |
| **P&L（損益）** | 未實現損益（綠色=盈利，紅色=虧損） |
| **Conf L/S（匯合度）** | 當前的多頭/空頭匯合分數 |

## 如何閱讀信號

### 決策流程
1. **查看背景色** → 淡藍色=HTF 多頭偏向，淡黃色=HTF 空頭偏向
2. **查看核迴歸線** → 藍線在橘線上方=上升趨勢，下方=下降趨勢
3. **觀察信號形狀** → 三角形=回調進場，菱形=均值回歸
4. **觀察標籤** → B1/B2/B3=分層買入，S1/S2/S3=分層賣出

### 交易範例（在 SOXL 5分鐘圖做多）
1. 背景變為**淡藍色**（HTF 1小時趨勢看多）
2. Kernel A（藍色）**上穿** Kernel B（橘色）— 趨勢轉換確認
3. 價格帶量突破上軌 → 動能脈衝已偵測
4. 價格回落至包絡線內部 → 回調進行中
5. 價格突破脈衝高點 → **綠色 ▲ 三角形**出現
6. 匯合分數達到 5+ → **B1（50%）**標籤出現 — 進場 50%
7. 分數升至 7+ → **B2（+25%）**標籤 — 加倉 25%
8. 價格回撤至核迴歸中線 → **Partial 25%** 標籤 — 部分平倉
9. 追蹤止損觸發 → **Trail ✕** 標籤 — 全部平倉

## 匯合度計分明細
系統每根K線為多頭和空頭方向各自計分（0–15分）：

| 因素 | 分數 |
|---|---|
| HTF 趨勢一致 | +2 |
| Kernel A 位於 B 的正確一側 | +2 |
| 首次回調信號觸發 | +3 |
| 一般均值回歸信號 | +2 |
| 強力均值回歸信號 | +4 |
| 價格在包絡線帶內 | +1 |
| 成交量高於平均 | +1 |
| **最高可能分數** | **15** |

### 分層門檻（可在設定中調整）
- **第一層（50% 進場）：** 分數 ≥ 5（預設）
- **第二層（+25% 加倉）：** 分數 ≥ 7（預設）
- **第三層（+25% 滿倉）：** 分數 ≥ 9（預設）

## 白色（中立區間）的交易策略

當背景是**白色（透明無色）**時，代表 1 小時線（HTF 大盤）正在說：「我現在很迷惘，沒有明確的方向。」

在這個白色區間，你應該這樣做：

### 1. 抓短線機會（打帶跑）
因為現在**沒有** 1 小時的大趨勢在背後撐腰，所以你不能期待價格會有一大波連續的行情。
* 如果出現了三角形（首次回調）或菱形（均值回歸）的信號，你還是可以進場。
* **但你要做的是：** 看到有賺就趕快跑！不要妄想這單能賺個大滿貫。只要帳上有利潤，就趕快把止損點往上移（保本），或者直接先獲利了結一半。

### 2. 等待「死亡交叉」或「黃金交叉」出現
當藍色線往下穿越了橘色線（或反過來），代表短線的趨勢發生了轉變。那個交叉點會跳出一個橘色的 **「Cross ✕」** 標籤。
* **你要做的是：** 即使背景還是白色的，這個交叉也代表短線表態，你可以開始尋找順著交叉方向的短線進場機會。

### 3. 讓系統的「計分板」幫你算數學
因為現在背景是白色的，我們的計分系統在算分的時候，**不會**給你那額外的 +2 分「大趨勢加分」。
* **你要做的是：** 如果系統在白色區域還硬是給了你一個 **B1 或 S1 標籤**，那就代表短線的機會真的太好了（其他所有的條件都非常完美），你可以跟著做，但請嚴格遵守標籤上的倉位大小（如 50%），不要貪心。
